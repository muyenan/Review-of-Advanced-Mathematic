# 高数复习网站

这是一个基于 `PPTs/tcu13ge_ppt_ch10.ppt` 到 `PPTs/tcu13ge_ppt_ch14.ppt` 章节结构整理的静态复习网站，覆盖：

- Chapter 10 无穷数列与级数
- Chapter 11 参数方程与极坐标
- Chapter 12 向量与空间解析几何
- Chapter 13 向量值函数与空间运动
- Chapter 14 偏导数

## 使用方式

现在可以直接双击 `index.html` 打开，不需要额外安装环境，也不需要先启动本地服务。

如果你仍然想通过本地地址访问，也可以在当前文件夹启动一个简单服务：

```bash
cd "/Users/qianmo/Desktop/大一下学习资料/高等数学/高数复习网站"
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000
```

## 站点特点

- 每章都有章节图解、公式墙、分节复习卡片、易错点和章末自测
- 参数曲线、极坐标、Taylor 逼近都带滑块或播放动画
- 空间向量、二次曲面、空间运动、切平面均支持拖动视角观察
- 图解优先使用 SVG，三维对象使用原生 Canvas 绘制
