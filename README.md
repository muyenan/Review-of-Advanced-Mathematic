# 高数复习网站

这是一个基于 `PPTs/tcu13ge_ppt_ch10.ppt` 到 `PPTs/tcu13ge_ppt_ch14.ppt` 章节结构整理的静态复习网站，覆盖：

- Chapter 10 无穷数列与级数
- Chapter 11 参数方程与极坐标
- Chapter 12 向量与空间解析几何
- Chapter 13 向量值函数与空间运动
- Chapter 14 偏导数

## 使用方式

现在可以直接双击 `index.html` 打开，不需要额外安装环境，也不需要先启动本地服务。

页面支持纯静态打开下的交互导航：

- 首页先展示五章总览，不再把全部内容堆成一个长页面
- 点击章节后进入单章阅读页
- 左侧全站导航栏支持手动收起和再次展开
- 所有交互图形、SVG 图解和 3D 画布都保留可用

如果你仍然想通过本地地址访问，也可以在当前文件夹启动一个简单服务：

```bash
cd "/Users/qianmo/Desktop/大一下学习资料/高等数学/高数复习网站"
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000
```

## 站点特点

- 首页提供五章总览卡片，每章先展示核心目标和小节目录
- 单章页按“章节图解、知识目录、公式墙、逐节整理、交互实验、章末自测”分层展示
- 左侧导航栏可以收起，收起后正文自动扩展
- 参数曲线、极坐标、Taylor 逼近都带滑块或播放动画
- 空间向量、二次曲面、空间运动、切平面均支持拖动视角观察
- 图解优先使用 SVG，三维对象使用原生 Canvas 绘制

## 文件说明

- `index.html`：页面壳子与静态入口
- `content.js`：五章知识点数据、公式和 SVG 图解内容
- `app.js`：页面渲染、章节路由、搜索与交互实验逻辑
- `styles.css`：整站排版、卡片层级、响应式布局与控件样式
